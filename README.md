# Big Box Exchange

Big Box Exchange is a mandate-first commercial real estate platform designed to connect acquisition teams, investors, developers, and agents through structured deal intelligence rather than traditional listings.

## 🚀 Vision
To become the operating system for commercial property deal flow — enabling verified participants to share mandates, portfolios, and opportunities in a secure, permission-driven environment.

## 🧱 Tech Stack
- Next.js (App Router)
- TypeScript
- Supabase (Auth, Database, RLS)
- TailwindCSS
